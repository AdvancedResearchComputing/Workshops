## Directories

On command bar at top of the landing page, click `Files` and click `Projects`.

You will be presented with a list of directories under `Projects`.

Find your project (can hit `cntrl-F` to search for it in the browser).

You can click on the name of the directory to go to it, 
or in the drop-down list to the right of the filename,
you can 
rename or download the directory.
You can also delete it.  Be careful.


Directories are shown with a yellow folder icon; files with a gray paper.

If you click on a file, the contents are shown on the screen.
In the drop-down list to the right of the filename, you can:
1. view
2. edit
3. rename
4. download
5. delete


![Example Directory Structure](./figures/ood-files-n-jobs/ood-directory.pdf)
[Example Directory Structure](figures/ood-files-n-jobs/ood-directory.png)

To go into a directory, double-click the directory name.

You can go up a directory quickly since the full path is provided above the
directory listing.
Also, there is an up-arrow to go up a directory.



## Accessing Terminal Windows on Clusters

Below the command bar in the graphic above, there are several white,
blue, and red buttons,
including `>_Open in Terminal`.  If you click that, you choose the
ARC cluster you wish to access.
Then a terminal window on that
cluster and in the directory that you are currently on in OOD
appears in a new browser tab.

## Create New File

On the same line as `>_Open in Terminal`, you can create a new file
with `+New File`.
You give the filename in a popup box and a new blank file appears in the
directory structure.
You can edit that file, as specified above.

The file will reside in whatever directory is currently active.


## Create Directory

You can create a new directory by clicking on `New Directory`.
Again, a popup will appear and you specify the new directory name.
It then appears in the directory structure as a subdirectory
of the current directory.


## Upload/Download Files

You can upload and download files with `Upload` and `Download` buttons.
You download whatever is highlighted in the directory structure below.

## For Each File or Directory

There is a drop-down list by each file.  You can rename, download, or delete.

## Copy/Move A Directory or File

You can copy and move files.
1. Highlight the file to move.
2. Click `Copy/Move` button.
3. Modify the path in the line of the ODD screen that is right below the `Copy/Move` button
to (i.e., navigate to) the directory where you want to copy or move the file.
4. On the left of the screen, click `Copy` or click `Move`.

Note this process is not good for making a copy of a file in a
directory where you want the copy in that same directory.
You might have to put the file in another directory, rename it, and move it back.
